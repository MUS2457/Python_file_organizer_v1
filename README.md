# Python File Organizer CLI

A simple Python CLI tool that organizes files inside a folder by sorting them into categories based on their file extensions.

 Features

 Scans a folder for files
 Detects file extensions
 Categorizes files (images, documents, videos, archives, etc.)
 Automatically creates folders for each category
 Moves files into the correct folder
  
## How It Works

1 The user runs the program.
2 The program asks for a folder path.
3 The folder is scanned for files.
4 Each file's extension is detected.
5 The extension is matched to a category.
6 Category folders are created if they do not exist.
7  Files are moved into their category folder.
